# N_Scanner
# 插件使用流程介绍
下载html文件可以在浏览器打开网页 
点击链接唤醒本地.bat文件 
在次批处理文件的执行过程中会调用上传的jar包（名为saomiao.jar），随后会自动调用本机dos（基于windows系统）扫描本地ip地址并生成 
此时需要输入ip.txt文件的目标地址以供后续扫描 
再调用nmap插件（可去官网下载）扫描本地端口生成结果并输出  

# 注意事项
本项目源码放在主分支的source文件夹下 格式为.class和.java 浏览者可以在本地配置好的java环境下打开
